ChartDirector 5.1 (Python Edition)

JsChartViewer.BottomCenter

Description

A constant equals to 2 to represent the bottom center position.

See Alignment Specification for supported alignment types.

Note: This is a static property (class property). It should be accessed as JsChartViewer.BottomCenter, not objName.BottomCenter.
